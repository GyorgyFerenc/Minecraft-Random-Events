## Source structure

``` Bash
src
├── main
│   └── java # Java sources
│       ├── antisocialgang
│       │   └── randomevents
│       │       ├── commands # For commands
│       │       └── RandomEvents.java # Main entry point
│       └── plugin.yml # Plugin file for the spigot API
└── test # Unit and other testing
    └── java
        └── antisocialgang
            └── randomevents 
target
│
...
└── randomevents-1.0.jar # The jar file of the plugin
```
